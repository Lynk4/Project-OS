# Warm Reboot

---
### Makefile is located in ```Project-OS/OS Dev - 2/bin```

---

### To build the OS

```
make OS
```

---

### Run OS.bin file 

```bash
qemu-system-x86_64 -boot a -fda OS.bin
 ```

---

### To delete all the .bin files

```bash
make clean
```
